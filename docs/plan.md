**Mitigación:** Cache Redis con TTL corto, webhooks redundantes, validación stock pre-pago### Riesgo 2: Fallos en Webhooks MercadoPago**Impacto:** Pagos no registrados, pedidos no asignados  **Mitigación:** Retry logic automático, polling backup cada 5min, alertas Slack### Riesgo 3: Escalabilidad DB**Impacto:** Slowdowns en horarios pico  **Mitigación:** Indexes optimizados, connection pooling, read replicas### Riesgo 4: Seguridad**Impacto:** Data breach, robo credenciales  **Mitigación:** Encriptación end-to-end, rate limiting, auditorías regulares, HTTPS only---## 🎓 Documentación
